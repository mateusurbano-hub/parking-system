# Roadmap 2025 – Sistema de Estacionamento

## Curto prazo (1–2 meses)

- [ ] Dashboard de faturamento por dia/mês
- [ ] Histórico de saídas com filtros (placa, período)
- [ ] Exportar relatórios em PDF

## Médio prazo (3–6 meses)

- [ ] Controle de mensalistas / parceiros fixos
- [ ] Log de auditoria (quem registrou entrada/saída)
- [ ] Tela de caixa com total do dia

## Longo prazo (6–12 meses)

- [ ] API REST para integração com outros sistemas
- [ ] Multi-estacionamento (vários pátios)
- [ ] Integração com OCR de placa em produção
